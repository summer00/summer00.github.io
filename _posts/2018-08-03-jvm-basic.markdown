# 内存模型

1. 线程隔离的数据区：
    * 程序计数器，指示当前线程所执行的字节码行号
    * 虚拟机栈，生命周期与线程相同。描述的是Java方法执行的模型：每个方法执行是都会创建一个栈帧用来存储局部变量表，方法出口，操作数栈，动态链接等，方法开始时入栈，结束时出栈。
    * 本地方法栈，与虚拟机栈类似，只不过执行的是Native方法

2. 线程共享的数据区：
    * 堆，存放对象实例
    * 方法区，存储虚拟机加载的类信息，常量，静态变量，及时编译后的代码等。包含运行时常量池
